Bulk Import Students

CSV format

Use a UTF-8 .csv file. The first row must be the header and use the column names below exactly (lowercase, underscores as shown). Extra columns are ignored.

Required (every row)

  • name — full name (stored as first + last name)
  • username — login id; must be unique in the system
  • password — initial password
  • class — class name; must match an existing class (grade) name in your school
  • roll_number — roll / register number

Optional

  • section — section letter/name; must match an existing section and be linked to that class, or it will be left blank

You may omit the section column entirely, or leave those cells empty.

Example header rows:
name,username,password,class,section,roll_number
name,username,password,class,roll_number (no section column)
